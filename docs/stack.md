# Technology Stack

## TanStack Start

Full-stack React framework providing:
- **Server-side rendering** (SSR) for fast initial loads
- **File-based routing** with type-safe route parameters
- **Server functions** for secure server-side logic
- Built on Vite for fast development

## Convex

Real-time backend-as-a-service providing:
- **Reactive database** — queries automatically update when data changes
- **Server functions** — type-safe queries and mutations
- **Authentication integration** — validates Clerk JWTs natively
- **Automatic indexing** — define indexes in schema, Convex handles the rest

## Clerk

Authentication provider offering:
- **Pre-built UI components** — SignIn, SignUp, UserButton
- **Session management** — secure cookie-based sessions
- **JWT integration** — issues tokens for third-party services like Convex
- **User management** — profiles, metadata, organizations

## shadcn/ui

Component library providing:
- **Copy-paste components** — you own the code, fully customizable
- **Radix UI primitives** — accessible, composable building blocks
- **Tailwind CSS styling** — consistent with CSS variable theming
- **New York style** — clean, modern aesthetic

## Tailwind CSS v4

Utility-first CSS framework with:
- **OKLCH color system** — perceptually uniform slate palette with subtle blue hue
- **CSS variables** — theme switching without rebuilds
- **Vite plugin** — fast compilation, no PostCSS config needed

## Geist Font

Vercel's typeface providing:
- **Geist Sans** — clean, modern sans-serif for UI text
- **Geist Mono** — monospace variant for code
- **Self-hosted** via `@fontsource` — no CDN dependency
